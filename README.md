# UI Assistant Clean

Um chatbox em **React** com integração de **Transformers.js** para rodar modelos de IA direto no navegador.  
Este projeto foi criado para explorar **UX/UI** e **IA local**, sem necessidade de backend.

---

## Funcionalidades
- Chat interativo com IA (GPT-2 rodando no navegador)
- Interface moderna com balões arredondados e avatares
- Tema claro/escuro com alternância
- Animação de digitação ("Digitando...")
- Scroll automático para última mensagem

---

## Tecnologias
- [React](https://react.dev/)
- [Transformers.js](https://xenova.github.io/transformers.js/)
- CSS Modules para estilização
